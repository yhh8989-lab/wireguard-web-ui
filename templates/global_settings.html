Wireguard Global Settings

Help

1. Endpoint Address
The public IP address of your Wireguard server that the client will connect to.
2. DNS Servers
The DNS servers will be set to client config.
3. MTU
The MTU will be set to server and client config.
4. Persistent Keepalive
By default, WireGuard peers remain silent while they do not need to communicate.
5. Firewall Mark
Add a matching fwmark on all packets going out of a WireGuard non-default-route tunnel.
6. Table
Value for the Table setting in the wg conf file.
7. Wireguard Config File Path
The path of your Wireguard server config file.
